Home Page Carousel

Now we can focus on building out the carousel. There are two main parts to this: the carousel itself, and the format of the shoutout in the middle.

The bubble for the shoutout is going to be tricky to style, especially with CSS only. We're going to start with making it rectangular, and push off the trapaziodal shape until later.

Lets first mock up what we think the shoutout could look like without the carousel.

  <div class="col-md-8">
    <div class="shoutout">
      <div class="body">
        <img src="http://placehold.it/85x85" class="avatar"/>
        <p class="shout">YOU ARE A SUPERHERO!!! THANKS FOR RESCUING MY EXCEL FILE!</p>
        <p class="author">by FRANK STEVENS</p>
        <p class="date">MAY 14, 2013 @ 3:47PM</p>
      </div>
      <div class="arrow"></div>
      <div class="shadow"></div>
    </div>
  </div>

(Sadly, there's no placekitten of the right size.)

Here's a first pass at styling the bubble:


$shout_image_size: 85px;
$shout_border_radius: 30px;
.shoutout {
  .body {
    background: #f9f9f9;
    border-radius: $shout_border_radius;
    padding: $shout_image_size - $shout_border_radius;
    margin-top: $shout_border_radius;
    margin-left: $shout_border_radius;

    img {
      float: left;
      margin-top: -$shout_image_size;
      margin-left: -$shout_image_size;
      border-radius: 100%;
      border: 2px solid #c9c9cb;
      height: $shout_image_size;
      width: $shout_image_size;
    }

    .shout {
      font-size: 30px;
      color: #01a84e;
      line-height: 1.5em;
    }

    .author {
      color: #999999;
      text-align: right;
      font-size: 18px;
      font-weight: bold;
    }

    .date {
      color: #0092ce;
      font-size: 10px;
      text-align: right;
    }
  }
}

I've created a few SASS variables to help organize the math. The avatar image is an 85px square, and we want to have it located over the top left of the bubble. This means that we need to position it up and to the left, and we need to adjust the margins and padding make sure that the text doesn't overlap. Since we know that this will eventually be inside of a carousel we need to make sure that none of the visual elements spill over the div. (Since the overflow will be cropped by the sliding functionality.)

Setting the background and border-radius of the shout bubble is straight forward. For the image, we position it absolutely, and shift it up the border radius. Instead of shifting it to the left, we adjust the margin of the shoutbox itself so that horizontally all of our visual elements are within the box.

We also set the border-radius to 100% on the image, which crops it as a circle.

Now that we've styled the main part, lets put in the arrow and the shadow.

.shoutout {
  .arrow {
    width: 0;
    height: 0;
    margin-left: 60%;
    border-top: 80px solid #f9f9f9;
    border-right: 70px solid transparent;
    padding-bottom: 35px;
  }

  .shadow {
    background-image: image-url( "shout_out_shadow.png" );
    height: 80px;
    width: 680px;
  }
}

Ways of making shapes using CSS directly can be found here: http://css-tricks.com/examples/ShapesOfCSS/

Making it a carousel

The carousel component is documented here: http://getbootstrap.com/javascript/#carousel

You need to pick an id for the carousel which is what links all of the controls together. The indicators reference this id using data-target and the next/prev reference the id using href.

First, lets factor out our shoutout code into a partial.

app/views/landing/_shoutout.html.erb:

<div class="shoutout">
  <div class="body">
    <img src="http://placehold.it/85x85" class="avatar"/>
    <p class="shout">YOU ARE A SUPERHERO!!! THANKS FOR RESCUING MY EXCEL FILE!</p>
    <p class="author">by FRANK STEVENS</p>
    <p class="date">MAY 14, 2013 @ 3:47PM</p>
  </div>
  <div class="arrow"></div>
  <div class="shadow"></div>
</div>

app/views/landing/_shoutouts.html.erb:

<%= render partial: "shoutout" %>
<%= render partial: "shoutout" %>
<%= render partial: "shoutout" %>
<%= render partial: "shoutout" %>

and then change the index.html.erb file to include a few partials, just to make sure that it loads correctly:

  <div class="col-md-8">
    <%= render partial: "shoutouts" %>
  </div>

Once you verify that this is working, what we're going to do is to put in the HTML components for the carousel. The carousel works by having an outer div with the class carousel, and then inside of it there are 3 main things:

  1. The carousel-indicators, which are the circles that show you which page you are on, which we don't need.
  2. The carousel-inner which contains all of the individual pages, which we are going to have as partials
  3. The carousel-controls which let you go from prev to next. Note that these should be outside of the carousel-inner div.

Let's get started, and edit the _shoutouts partial. (Notice that one of the partials is signular, and one is plural. We are editing the plural one.)

<div id="shout-carousel" class="carousel slide" data-ride="carousel">
  <div class="carousel-inner">
    <div class="item active">
      <%= render partial: "shoutout" %>
    </div>
    <div class="item">
      <%= render partial: "shoutout" %>
    </div>
    <div class="item">
      <%= render partial: "shoutout" %>
    </div>
    <div class="item">
      <%= render partial: "shoutout" %>
    </div>
  </div>

  <!-- Controls -->
  <a class="left carousel-control" href="#shout-carousel" role="button" data-slide="prev">
    <span class="glyphicon glyphicon-chevron-left"></span>
  </a>
  <a class="right carousel-control" href="#shout-carousel" role="button" data-slide="next">
    <span class="glyphicon glyphicon-chevron-right"></span>
  </a>
</div>

And load it up!

It works! And looks terrible!

What needs to change:

  1. Get rid of the control gradient
  2. Fix the chevrons
  3. Put in some right and left margins so that the controls don't overlap the shoutout
  4. Fix it so the top of the shout lines up with the top of the signin form.

Lets look at the SASS source file for bootstrap carousel, https://github.com/twbs/bootstrap-sass/blob/master/assets/stylesheets/bootstrap/_carousel.scss

Scrolling through this file we can see which things can be customized and changed. In our scss file, lets turn off the gradient background for the control.

a.carousel-control.left, a.carousel-control.right {
  background: transparent;
}

What else is in there? $carousel-control-width seems interesting, lets give that a try.

Lets first move the $shout_image_size: 85px; and $shout_border_radius: 30px; variables up to the top of the CSS, above where you @import "bootstrap";. This is a good idea for two reasons, keeping all the variables in one place makes it easy to see where everything is, and having it before the import will let you override some of bootstraps configuration. Lets do this now for the carousel-control-width

$shout_image_size: 85px;
$shout_border_radius: 30px;
$carousel-control-width: 30px;

@import 'bootstrap-sprockets';
@import 'bootstrap';

Now lets add some margins to the .shoutout, but only when it's in the carousel.

.carousel .shoutout {
  padding-left: $carousel-control-width;
  padding-right: $carousel-control-width*2;
}

Finally, let's swap out the old glyph icons style and put in our slightly narrower image. We'll also need to move up the arrows a bit to make it match with our design.

in app/views/landing/_shoutouts.html.erb

      <!-- Controls -->
      <a class="left carousel-control" href="#shout-carousel" role="button" data-slide="prev">
        <span class="icon-prev leftarrow"></span>
      </a>
      <a class="right carousel-control" href="#shout-carousel" role="button" data-slide="next">
        <span class="icon-next rightarrow"></span>
      </a>

in app/assets/stylesheets/application.css.scss

.carousel-control {
  color: transparent;

 .leftarrow {
    background-image: image-url("left_arrow.png");
    height: 49px;
    width: 19px;
    margin-left: -10px;
  }

  .rightarrow {
    background-image: image-url("right_arrow.png");
    height: 49px;
    width: 19px;
    margin-right: -10px;
  }

  .icon-next, .icon-prev {
    top: 35%;
  }

  .icon-next::before, .icon-prev:before  {
    content: "";
  }
}

Looking good! Now lets style the signed in homepage.
