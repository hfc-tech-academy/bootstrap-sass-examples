Signin button

Lets start from the top. Lets change the overall style of all the sites buttons, as specified with the .btn class in app/assets/stylesheets/application.css.scss:

.btn {
  border: 3px solid #ededed;
  color: #bababa;
  @include gradient-vertical($start-color: rgba(255,255,255,1), $end-color: rgba(220,220,218,1));
  box-shadow: inset 0 0 2px 2px #ededed, 0 1px 1px 1px #888;
}

This creates a border that is 3 pixels wide and offwhite, changes the color of the text, and adds a vertical gradient to the background of the button. The @include gradient-vertical is a SASS mixin that bootstrap provides which lets you include parameterized rules into your CSS. You pass in the starting and ending colors, and the bootstrap code will compute the rules that are needed. If we look at the generated rules, this is what is produced:

.btn {
    border: 3px solid #ededed;
    color: #bababa;
    background-image: -webkit-linear-gradient(top, white 0%, #dcdcda 100%);
    background-image: -o-linear-gradient(top, white 0%, #dcdcda 100%);
    background-image: linear-gradient(to bottom, white 0%, #dcdcda 100%);
    background-repeat: repeat-x;
    filter: progid: DXImageTransform.Microsoft.gradient(startColorstr='#FFFFFFFF', endColorstr='#FFDCDCDA', GradientType=0);
    box-shadow: inset 0 0 2px 2px #ededed, 0 1px 1px 1px #888;
}

All of those background-image rules were generated from the mixin, and are for webkit (Safari + Chrome), Opera, CSS3 Standard, and IE9 and down. Bootstrap has figured all of this out for you, so luckily you don't need to.

The code for this (and others) can be found at https://github.com/twbs/bootstrap-sass/blob/master/assets/stylesheets/bootstrap/mixins/_gradients.scss

The hover state needs a little work, so lets flesh it out a little

.btn {
  border: 3px solid #ededed;
  color: #bababa;
  @include gradient-vertical($start-color: rgba(255,255,255,1), $end-color: rgba(220,220,218,1));
  box-shadow: inset 0 0 2px 2px #ededed, 0 0px 1px 1px #0b6184;

  &:hover,
  &:focus,
  &:active,
  &.active,
  .open > &.dropdown-toggle {
    color: darken(#bababa, 10%);
    border-color: #bababa;
    @include gradient-vertical($end-color: rgba(255,255,255,1), $start-color: rgba(220,220,218,1));
  }
}

Notice another SASS function here darken. This takes a color and darkens it by a certain percentage.

Signing Button Icons

Lets change the html to include the button icons. We'll pull them from the glyphicon set, which is documented here: http://getbootstrap.com/components/#glyphicons

<button class="btn btn-default signin" type="button">
  <span class="glyphicon glyphicon-home"></span>
  <span class="text">Sign In</span>
  <span class="glyphicon glyphicon-chevron-down"</span>
</button>

We've also added a signin class to the button, so we can target the styles directly for this element. Lets go ahead and change the color, and add some padding after the text:

.btn.signin {
  color: #0b90c9;
  margin-top: -3px;
  font-size: 18px;
  .text {
    padding-left: 6px;
    padding-right: 100px;
  }
}

This changes the color, slides the top of the button off the screen, and adds some padding around the button text.

Fonts

At the top of app/assets/stylesheets/application.css.scss lets import the font what we want to use on the site. We are going to set a bootstrap variable to use this at the main font of the site.

@font-face {
    font-family: 'lubalinbook';
    src: font-url('lubalinbook-webfont.eot');
    src: font-url('lubalinbook-webfont.eot?#iefix') format('embedded-opentype'),
         font-url('lubalinbook-webfont.woff') format('woff'),
         font-url('lubalinbook-webfont.ttf') format('truetype'),
         font-url('lubalinbook-webfont.svg#lubalinbook') format('svg');
    font-weight: normal;
    font-style: normal;

}

@font-face {
    font-family: 'lubalinmedium';
    src: font-url('lubalinmedium-webfont.eot');
    src: font-url('lubalinmedium-webfont.eot?#iefix') format('embedded-opentype'),
         font-url('lubalinmedium-webfont.woff') format('woff'),
         font-url('lubalinmedium-webfont.ttf') format('truetype'),

$font-family-base: 'lubalinbook', "Helvetica Neue", Helvetica, Arial, sans-serif;

@import 'bootstrap-sprockets';
@import 'bootstrap';

$font-family-base is the bootstrap variable that lets you reconfigure what family of fonts that boot strap uses. The major variables are defined at https://github.com/twbs/bootstrap-sass/blob/master/assets/stylesheets/bootstrap/_variables.scss
