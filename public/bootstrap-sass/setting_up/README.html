Bootstrap-sass on Rails

Setting Up

Setting up the bootstrap is pretty straightforward. We are going to be using the gem found at https://github.com/twbs/bootstrap-sass

We outline two methods: the first is to create an app, add the gem to the Gemfile and insert a few links inside of the assets directory to get things going.

The second is to use the HappyFunCorp seed application generator to automate all of these steps. If you are working on an existing application, you should use the direct gem install. (It's probably good to go through it at least once anyway to make you know what's going on.) If you are starting from a new project, the seed generator will automate a lot of the process which will let you focus on prototyping quickly.
