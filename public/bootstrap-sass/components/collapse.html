Collapse

The collapse component lets you toggle between displaying content and hiding it. To use this, you specify data-toggle="collapse" and then a data-target="#element_id" to control which element is shown or hidden.

<a class="btn btn-primary" data-toggle="collapse" data-target="#moreInfo">Show me more!</a>

This is similar to the dropdown component, which is mainly for menus. This is more general, doesn't (necessarily) require any specific HTML elements, and will animate the transition.

.collapse will hide the content.

.collapse.in will show the content.

.collapsing is added with the transition begins.

Collapse can be used across multiple components, so that only one of them is open at a given time. This is done by adding a data-parent to them if you use this in conjuction with the panel classes, you can create accordians with this, as shown in the official documentation.
